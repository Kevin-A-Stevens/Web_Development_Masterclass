Cascade

Process of combining different stylesheets and resolving conflicts between different CSS rules and declarations, when more than one rule applies to a certain element

  • Importance (weight)
  • Specificity (inline, IDs, Classes, Elements)
  • Source order

Inheritance

Not all properties are inherited.

padding is not inherited, but line-height is

properties related to text are inherited, such as font-size, font-family, color, etc

Converting px to rem, An effective workflow

  • set font-size to 10px percentage in html element
  • 10 / 16 = 0.625 = 62.5%
  • You can now use rem such as 16px = 1.6rem. 20px = 2 rem

Component driven design

  • Think about the components that will make up your design. They should be reusable across the project(s)
  • Build using BEM (Block Element Modifier)
  • Block = stand-alone component such as .block
  • Element = Has no meaning on it's own such as .block__element
  • Modifier = Makes a different version of an element, such as .block__element--modifier
  • Architect by organizing your code using 7:1
  • 7 folders for partial SASS files and 1 main SASS file
  • 7 folder: base, components, layout, pages, themes, abstracts, and vendors
