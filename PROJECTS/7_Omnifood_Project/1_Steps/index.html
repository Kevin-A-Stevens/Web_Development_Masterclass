Development steps

Define

  • Who is the website for
  • What is the main purpose (Business/User goals)
  • Define target audience

Plan

  • Gether website content (usually provided)
  • Plan sitemap (pages)
  • Plan sections on each page
  • Define website personality (See Web Design)

Sketch

  • Think about components you need
  • Sketch ideas with pen and paper or Figma
  • Experiment with different ideas

Design and Build

  • Design using HTML/CSS
  • Design visual styles
  • Use client's branding whenever possible

Test and Optimize

  • Does website work well in all major browsers
  • Does website work on mobile devices
  • Optimize all images in dimensions and file size
  • Fix accessibility problems
  • Run the lighthouse performance test in Chrome
  • Think about Search Engine Optimization

Launch

  • Upload to a hosting platform (netlify)
  • Buy a great domain name

Maintain and Update

  • Keep website current
  • Install analytics software (Google Analytics/Fathom)
  • Keep a blog updated
